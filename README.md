Utilities for JS projects. **/^o^/**

## Usage
### Installation
```
npm/pnpm install pantsu
```

### Importing and usage
```ts
import { StringUtils, ... } from "pantsu";

StringUtils.equalsIgnoreCase("str", "str");
```

## Setup & Build
Install all dependencies:
```
pnpm install
```

Build:
```
pnpm run build
```
